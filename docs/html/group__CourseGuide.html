Detailed Description

Course Guide Overview

The Course Guide introduces the structure, workflow, and documentation model used throughout the C++ Course Template. It explains how students and instructors interact with the repository and provides a unified entry point to all topics.

This guide is organized into the following sections:

  • Repository Setup
  • Directory Structure
  • Local Workflow
  • GitHub Workflow
  • CMake Integration
  • Unit Testing
  • Coverage and Reports
  • Documentation Generation

How to Use this Guide

  • Start with Repository Setup (CG_RepositorySetup) to fork and clone the repository.
  • Continue with Directory Structure (CG_DirectoryStructure) to understand the layout.
  • Read Local Workflow (CG_LocalWorkflow) to build, run, and test the project locally.
  • Check GitHub Workflow (CG_GitHubWorkflow) to see how CI validates your work.
  • Explore Unit Testing and Coverage to understand quality gates for your code.
  • Finally, use Doxygen (CG_Doxygen) to learn how documentation is generated from comments and guide files.

This page is meant to be your main hub: from here you can jump into any aspect of the course infrastructure and workflow.
